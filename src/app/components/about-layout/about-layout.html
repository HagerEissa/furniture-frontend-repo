About Our Furniture Website

Welcome to Furniro — a modern furniture shopping experience created with passion and technology.

Our Project

This furniture e-commerce platform was developed by a team of five web developers using Angular for the frontend, Express.js and MongoDB for the backend, and OAuth 2.0 for secure authentication. It also integrates Cloudinary for image management.

On the frontend, we built the UI using Angular and Bootstrap to ensure a responsive and elegant design. On the backend, we used Node.js with Express and MongoDB as our database. We implemented major packages like Stripe for handling payments, Cloudinary and Multer for image uploads and management, and OAuth 2.0 for enabling login via Google and Facebook.

The admin panel provides full CRUD functionality for managing users and products, giving admins complete control over website content. Regular users can browse products, edit their profiles, add items to cart or favorites, place orders, and even leave reviews and comments — all while enjoying a secure and smooth experience.
